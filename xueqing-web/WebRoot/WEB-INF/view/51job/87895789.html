职位信息

专业技能：
1. 熟练掌握 Java 基础知识，代码编写规范，熟悉主流的 java 设计模式；
2. 熟练掌握 JSP、Servlet 等 JavaWeb 技术，熟悉 HTTP 协议和 JDBC 编程；
3. 熟练掌握 Javascript 编程和 jQuery、jQuery、EasyUI 等前台框架等前台框架；
4. 熟练掌握 Struts 2、SpringMVC Web 层框架及其特性，熟悉 Mybatis、Hibernate 数据访问层框架；
5. 熟练掌握 Spring 框架，包括 IOC 和 DI、AOP 编程、声明式事务处理及 SSH、SSM 的整合技术；
6. 熟悉 MySql，Oracle 等关系型数据库，熟悉 plSql 语法和 sql 语句，掌握 redis 常用操作。

岗位职责：
1、负责智能建筑软件应用的需求调研、分析和方案设计、文档编制等，负责智能建筑的信息化管理、智慧化应用等产品与服务的开发，涉及系统集成、联动控制、安全运维、决策支持、能耗监测、能效分析以及基于云计算、物联网、大数据、移动互联网等技术的整合创新，负责制定软件系统的实施方案，并根据需要进行技术指导；
2、负责跟踪研究智能建筑软件应用的前沿技术，搜集、整理智能建筑相关行业现状和趋势，分析竞争对手的功能与特色，进行用户数据分析并找到对应提升方案，了解智能建筑信息化相关的标准、协议等动态，为公司决策提供依据；
3、其他需配合、支持完成的工作；
4、进行编码开发和搭建云平台。

任职要求：
1、本科以上学历，计算机及相关专业，有大型智能建筑应用软件系统开发经验，3年以上网站架构设计经验；
2、专业基础扎实，具有独立的应用软件研发能力，熟悉主流的软硬件技术架构、开发平台、数据库技术和互联网应用开发，掌握常用的设计模式和系统建模工具，掌握智能建筑软件接口技术，对软件工程标准规范有深入理解和实践；
3、职业素质优秀，责任心强，具有很强的执行力、抗压力、沟通能力、文案能力以及团队精神和钻研精神。

职能类别： 互联网软件开发工程师 软件工程师
