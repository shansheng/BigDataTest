软件研发总监

上海 3-5万/月

上海经达信息科技股份有限公司

该公司所有职位

民营公司   |  50-150人   |  计算机服务(系统、数据服务、维修),交通/运输/物流

申请职位 收藏 竞争力分析
5-7年经验 硕士 招1人 02-06发布
英语良好 计算机科学与技术 软件工程

五险一金 餐饮补贴 年终奖金 绩效奖金

职位信息

一、岗位职责：
1、全面管理公司技术研发和项目管理工作,带领团队完成公司制定的产品研发目标；
2、规划公司的技术发展路线与新产品开发，及时了解和监督技术发展战略规划的执行情况；
3、负责重大技术项目的决策及方案审核，指导并参与核心代码的书写，组织解决项目开发过程中的重大技术问题；
4、有效带领技术团队，监督及指导技术部门的工作，保证项目顺利落地，建立并完善公司各项研发规范及流程；
5、了解行业新技术，新应用，并应用到公司产品研发和日常管理中；
6、与其他部门协同沟通完成公司产品项目研发目标。

二、任职资格：
1、计算机科学技术专业毕业，硕士以上学历；
2、在有一定规模的软公司（包括海外）从事过5年以上的专业工作，担任过软件研发总监或者有30人以上的软件开发队伍的实际负责人的经历，有丰富的软件队伍组织管理、协调经验；
3、精通C/C++/JAVA/PHP等主流体系架构之两种架构和设计模式，熟悉大数据和云计算相关技术理论，具有较强的架构和设计经验，对现有大数据、云计算主要产品的相关技术有深刻研究和了解，；
4、精通数据库设计、分析、代码编写和调试，熟悉MySql/SqlServer/Oracle三种数据库架构中任一种的具体操作；
5、强大的系统分析能力，熟悉UML或者其他流行的系统分析工具之一；
6、有智能交通领域大数据处理、实时监控以及UBI车险技术方面经验者优先，有MongoDB / Hadoop / Hbase / Cassandra / Storm / Spark / Hive 等系统的开发经验者优先；
7、强大的代码编写能力，以及对于开发部门员工的核心能力提升的培训能力。

职能类别： 技术总监/经理 高级软件工程师

关键字： 软件研发 UBI 研发总监

公司信息

上海经达信息科技股份有限公司为上海市人民政府批准成立的注册资本人民币5449万元，聚焦于智能交通系统投资、研发、生产及系统集成于一体的高科技企业。主营业务为公路交通工程机电系统工程承包及智能交通信息管理系统总承包 公司具备国家建设部核发的《公路交通工程专业承包通信、监控、收费综合系统工程》资质以及国家工信部颁发的《计算机信息系统集成》等资质。经达公司是经上海市人民政府核定的“上海市高新技术企业”、“上海市软件企业”，并已通过了ISO9001/2000质量体系论证。2006年底通过上海市科委、经委联合评审，本司光荣地成为上海市首批50家“科技小巨人培育企业”之一,现已通过专家验收。2010年被评选为第五届上海市科技企业创新奖及上海市首批“专精特新”企业。
    公司在道路智能交通领域取得了较好业绩。公司开发的城市道路交通信息诱导系统被专家评审为具有“国内领先”水平，其核心算法及数学模型被国家权威部门评定为具有“国际领先”水平。公司开发的采用视频检测及模式识别技术的“行人闯红灯自动监控报警系统”被评定为国内首创，并得到了包括中央电视台及《新民晚报》等新闻媒体的大力报导。 公司作为国内从事公交智能调度及监控管理的主要集成商之一，其提供的车载GPS终端通过了国家权威机构的检测。公司能提供全套公交调度和监控软件、公交视频监控软件、公交行业管理软件、公交行业政府监管软件、公交辅助决策软件等，是国内能提供全套公交行业软、硬件的少数供应商之一。
公司主页：http://www.sssjd.com
