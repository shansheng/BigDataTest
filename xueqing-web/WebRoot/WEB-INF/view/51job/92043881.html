Title: Cloud Application Developer (full stack engineer) _微软人事外包职位

Microsoft IoT Insider lab is looking for an experienced full stack developer to join the team in Shenzhen.
The full stack engineer will work with customers and partners and will be responsible for guiding customers in the design, engineering and implementation of Microsoft Azure and Microsoft Azure IoT based solutions. This position involves working with customers to design, code, debug/troubleshoot and otherwise guide to completion their IoT projects or solutions. A successful candidate will possess the following skills and capabilities:

Tech
？ Well versed in connectivity, event processing, cloud storage and transform technologies
？ Well versed in utilizing connectivity protocols HTTP and others such as AMQP or MQTT. Low level understanding of TCP/UDP as well as DNS routing and IP tables a plus.
？ 3+ years of experience building web-based applications using two or more C#, PHP, Python, Java or Node.js.
？ Previous experience in development on a public cloud (Amazon Web Services, Ali Cloud, Azure)
？ Experience with one or more types of data storage, SQL Server, MySQL, MongoDB or other NoSQL database

Skills
？ Experience building and deploying cloud applications
？ Experience with authentication and authorization such as AD or OAuth
？ Knowledge of the private cloud and hybrid cloud configuration
？ Knowledge of standard security topics related to web-based services
？ Experience developing both client and server sides of a connected device solution
？ Previous experience in developing code on device to connect to a public cloud, storing the data and analyzing it in the cloud
？ Familiarity with private cloud and hybrid cloud configurations
？ Experience administrating servers remotely in a cloud environment
？ Familiarity with Windows Azure
？ Proficiency using standard development tools, Git or other DVCS, debuggers, issue trackers like Jira, as well as CI tools like Travis or Jenkins.
？ Familiarity with standard coding practices such as code reviews, unit testing, integration testing, scrum, agile etc.
？ Strong communications (written and speaking) and interacting with customer and partners to build their solution, trouble shoot roadblocks
？ Excited about doing full stack development
？ Strong organizational, time management, communication, and engineering skills are necessary

The successful candidate will have a BS engineering degree or equivalent work experience combined with a minimum of 8+ years relevant work experience. We are looking for experienced individuals located in Shenzhen to be a part of this exciting team.

职能类别： 软件工程师
