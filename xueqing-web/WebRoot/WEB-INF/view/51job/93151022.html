职位信息

招聘要求：
1、技术员至少大专以上学历，一年以上相关工作经验；
     助理工程师要求本科以上学历，有相关工作经验***。
     熟悉打线、焊接设备的参数调试、具备基础的失效性分析，能够熟练使用excel办公工具，具有一定的动手能力，勤于思考，工作积极 主动。
2、技术员：18-30岁，身体健康，视力良好，女生优先，心灵手巧，能在显微镜下手动操作微小的光学组件，有光器件耦合、贴片、打线经验的优先；
3、能够适应无尘实验室作业环境。

薪酬福利

1. 工作环境―环境舒适，咖啡厅，茶水间，休息室一应俱全。每天早上和下午都有水果和糕点。五天八小时工作制。
2. 工资待遇―待遇丰厚，每年至少有一次加薪机会，表现突出者还可享受绿色通道。
3. 生活环境―为有需求的员工提供宿舍，补贴部分午餐餐费。
4. 休闲娱乐―设有篮球、足球、羽毛球和游泳等协会；不定期组织娱乐活动；每年组织至少一次省内外旅游；年底举办大型文艺晚会及抽奖活动。
5. 社保福利―按国家规定，入职即为员工缴纳五险一金。
6.假期福利―所有员工均可带薪享受国家规定的各类假期，包括法定节日、年休假、婚假、产假、病假、工伤假、丧假等。

四、公司简介

易飞扬是全球光网络器件领域的市场创新者和应用技术开拓者。我们研发与制造革新技术的基础光网络器件和光互联产品，广泛用于固定电信网络,移动通信网络，数据通信网络，区域存储网络，大数据和云计算中心，视频监控与传输，工业视觉，卫星通信，消费类光通信等领域。
六大产品线
基础光器件：
有源光器件：155M-10G 可插拔光收发模块、10G 可调XFP/SFP+光模块，3G/18G 射频光模块等
无源光器件：光波导分路器、CWDM/DWDM分合波器，50G/100GHZ阵列波导光栅,微机电光衰减器等
光互联产品：
高速光模块：25G SFP+,40G/100G QSFP，100G CFP/CFP2/CFP4，120G CXP等
高速线缆: 10G/25G/40G/56G/100G 电缆组件和有源光缆组件等
视讯产品: 视频 SFP光模块，HDMI 光延伸器等.USB3.0 有源光缆，HDMI 有源光缆等；
物理布线：数据中心云计算机房MPO/MTP高密度主干光缆组件，MPO/MTP扇出光缆组件等；
全球业务发展
易飞扬总部位于中国深圳。在北美硅谷设有分公司，在俄罗斯设有代表处。在巴西和以色列有独家代理商。在美国和欧洲拥有一些重要的渠道分销合作伙伴。
2014年易飞扬的销售额为5000万美元左右，预计在未来的3年内业绩可以达到1亿美元，在未来5年内可以达到2亿美元，跻身成为全球一流光器件与光互联产品供应商。
当前，易飞扬海外业务占公司总体业务的70%，国内业务占30%。国内主要的业务聚焦于如百度、阿里等互联网运营商，中国联通、中国电信等传统业务运营商以及相关数据通信设备商。
海外业务主要聚焦于北美，俄罗斯，巴西，以色列，印度等地的电信和互联网运营商、设备商等。
经过数十年经营，凭借良好的市场形象和产品口碑，易飞扬（GIGALIGHT）品牌在全球获得卓越的成长和广泛认知。
 
公司里程碑
2006-2009 致力于10G速率以下光器件研发生产和销售，其中10G 光模块为公司全球业务发展奠定了基础
2010-2012 进入光无源产品领域，同期投资研发40G/100G 光互联核心技术平台
2013-2014 完成有源光缆在数据中心应用的商业化进程。推出USB/HDMI有源光缆、光延长器等产品，进入 视讯和消费类光通信市场
2015-2016 致力开发100G光模块，进入USB/HDMI有源光缆和光延长器领域，围绕数据中心开发100G DAC和 100G有源光缆，开发云编程器、系列智能型光模块测试仪、开发6G/12G SDI视频光电模 块，INFIBERONE 电商平台丰富GIGALIGHT光模块生态圈。

2017-- 建立武汉研发中心，布局100G 长距离光模块，200G /400G高速光模块，相干通信和硅光子技术。

技术能力
易飞扬的R&D 目前有约100名工程师，占公司总人数的20%。其中研发团队多为硕士以上学历。截止2015年6月，团队已经获20多项发明专利授权。易飞扬目前在40G/100G 多模光纤阵列技术平台和集成光器件领域处于国内领先水平和国际一流水平，主要的能力是可以自制高速平行光引擎并申请获批了多项专利。新时期的亮点产品与技术包括USB3.0和HDMI有源光缆，光载无线（18GHZ)模块。易飞扬的下一步研究的重心一方面将重点打造低功耗与更智能的数据中心和云计算光互联模块与线缆，另一方面将重点投入视讯类光互联产品开发，迎接即将开启的视讯高清时代。同时公司也将密切跟踪SDN技术，研究软件定义的光模块与云终端。

武汉分公司致力于高速光模块与光器件研发为主。欢迎光通信领域优秀人力加入，合力发展，共创飞扬辉煌！
