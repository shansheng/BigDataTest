职位信息

一、招聘要求：
1、2018届应届本科生，计算机科学与技术、软件工程、通讯工程、电子信息工程等计算机相关专业或管理信息系统、电子商务、市场营销、工商管理等管理类专业；
2、熟练掌握Office办公软件，了解SQL Server数据库、Windows Server操作系统等优先；
3、沟通能力较佳，具有较强的计划、组织和协调能力，有较强的责任心和服务意识，身体健康，能适应出差。
二、岗位职责：
1. 辅助项目经理负责项目实施过程中对调研收集、整理客户的需求资料，协助编制系统需求分析说明书；
2. 辅助项目经理负责项目实施工作的具体推进，管控项目实施进程，并控制客户需求及变更的管理等；
3. 负责项目的质量掌控、参与软件的测试、客户培训、系统上线辅导、验收等相关实施工作；
4. 负责对客户软件使用问题的进行维护和业务咨询工作。

工作地点：广东珠海，也可在培训后按照员工个人意愿派驻上海、武汉、广州、南宁、成都、济南等分公司/办事处工作。
薪资待遇：年薪6-10万。
晋升通道：
实施序列：实施经理--项目经理—高级项目经理—项目部（区域）负责人
营销序列：项目经理-高级业务经理—分公司（办事处）负责人
产品序列：高级项目经理—产品经理


三、公司简介
珠海市世纪信通网络科技有限公司是一家国家高新技术企业、双软认证，工程建设企业“互联网+”综合服务商，致力于通过移动互联网、软件、云计算、大数据、BIM、智能硬件等新技术，助力建设行业提升企业运营效率，提升项目管理水平。15载潜心研究与发展，专注工程建设细分行业，公司在建设行业信息化领先已拥有领先地位，公司产品“监理通”已成为国内监理企业信息化首选品牌和市场占有率第一。
公司设立产品中心，拥有自主知识产权iBOSS智能开发平台、移动APP开发平台,公司的产品和项目曾多次获得国家、省级的管理创新、创新基金资助和奖励，10多个软件产品获得高新技术产品认证和软件著作权。
公司依托珠海研发基地，不断完善营销、服务体系，已在华南、华东（上海）、西南（成都）、中南（武汉）、华北（济南）等建立了分公司（办事处），为客户提供零距离的优质服务。
为顺应公司高速发展需要，我们诚邀有责任感、学习能力强的各类优秀毕业生加盟并与公司共同发展。

主营产品：
监理通：监理企业综合业务管理系统，监理行业信息化的首选品牌
项管通（C3PM）：大型建设项目管理信息平台
施管通（C3E）：施工企业综合业务管理系统

公司信息

珠海市世纪信通网络科技有限公司（简称“世纪信通”），一直从事工程项目管理和项目型企业管理相关领域的方法、技术研究和软件开发，主营业务包括：
1）致力于提供具有自主知识产权的企业级项目管理软件、服务与解决方案；
2）互联网增值业务运营管理。
项目管理已经成为现代企业管理的三大支柱，尤其对于项目型组织。项目型组织的业务，就是一个个项目—启动、执行、管控、完成。每一个项目的成败，极大的影响到组织的业绩，甚至决定了组织的生死存亡。现代企业迫切需要以项目管理为导向，进行战略实施管理，确保战略形成和实施的效率，提高企业的核心竞争力。
以项目作为组织经营、管理、资源分配的最小单元，并从整个组织来规划、提升项目管理能力，是项目型组织的管理趋势，是组织级项目管理的核心，也是珠海市世纪信通项目管理软件产品的设计思路。
珠海世纪信通公司拥有多名知名项目管理专家，以及PMI认证的国际项目管理师（IPMP），为我们的客户进行项目管理体系的咨询与诊断，将先进项目管理技术与客户需求有机结合，借助专业的项目管理软件产品，从而帮助客户的决策层、管理层、项目执行层，更好的管理项目。
依靠领先的技术、高水平的产品、强大的咨询实施队伍、优秀的本地化服务及交付能力，珠海世纪信通公司的项目管理软件已经在通信建设、建筑、石化、新产品研发、系统集成、IT建设/监管、专业咨询服务等行业得到了广泛的应用。
企业的宗旨： 与用户真诚合作，做用户可靠朋友
核心价值： 帮助客户构建企业项目管理体系，打造项目管理方法论
企业理念： 通过为顾客带来价值，为股东带来价值增值，为员工带来价值实现，把公司发展成为专业的项目管理服务提供商。
企业文化核心： 以人为本、客户为本
产品方针： 实用、先进、可靠
服务方针： 专业、及时、真诚
质量方针： 实用、先进、可靠的产品，专业、及时、真诚的服务
