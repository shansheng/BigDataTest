职位信息

岗位职责：
1.基于海量用户行为数据和其他数据，分析和研究数据与实际业务的关联关系，开发设计可扩展的机器学习算法，挖掘用户属性和行为特征，并以实际业务应用为导向；
2.参与数据清洗和过滤、特征抽取、参数选择、算法实验、效果分析；
3.整理并分析用户数据，撰写分析报告，从数据的角度给出运营建议；
4.业务的策略优化及算法研发，提供先进、适用的解决方案（模型）；
任职要求：
1.2年以上Python相关开发经验，掌握Django／flask等开发框架，有数据挖掘经验的优先考虑；
2.熟悉各种Linux命令的使用，能够在Linux环境下进行开发工作，对云计算技术有一定的了解；
3.熟悉mysql数据库，具备mysql、mongodb、redis等开发经验；
4.有较强的分析问题与独立解决问题的能力，良好的沟通能力及团队合作精神；
5.有互联网产品开发经验或分布式系统设计经验优先考虑。

职能类别： 互联网软件开发工程师

公司信息

公司简历：
    广州棒谷科技股份有限公司，成立于2009年，旗下有广州霆辉贸易有限公司/广州芊妮贸易有限公司。总部位于广州，在深圳、东莞、长沙、杭州、义乌均设有分公司，拥有精英团队超过 3000人的年轻精英团队。专注于B2B、B2C跨境自营平台产品销售，并且深耕于Ebay\Amazon\AliExpress\Wish\Newegg 等第三方平台，以客户需求为主导，自主开发新品，提供个性化购物体验。公司营销网络遍布全球一百多个国家，产品主要出口英美德法、西班牙、澳大利亚、俄罗斯等国家，并积极进入中东、东南亚等新兴市场。在多方平台的共同推进下，棒谷现已成为当前中国B2C对外贸易行业内的佼佼者。

业务发展：
    棒谷在2009年开通的自营综合类商城网站（www.banggood.com），不断开拓新品，秉承“贴近客户需求，提供优质服务”的宗旨，坚持物美价廉的原则，为海外顾客提供优质的中国产品，注册用户遍及全球一百多个国家，日访问量超过100万。现主营产品有20多个种类，共计10万多种商品，如：手机、电脑、数码配件、摄影摄像等电子产品、 玩具航模类、户外运动休闲类，婚纱婚礼用品、服装、鞋靴、箱包类，家居、家纺、厨卫等类别，产品性价比高，深受海外商家/消费者青睐。
    目前Ebay\Amazon \AliExpress\Wish\Newegg等第三方平台正在迅速发展，目前的主要业务覆盖全球，由英美市场，逐步扩展到德法、俄罗斯、澳大利亚，以及北欧等市场，共计上线产品50000多种商品，产品性价比高，质地优良，深受海外上商家的青睐，在电子类、fashion类产品等领域占有一定的服务优势。
    随着业务不断扩大和发展，新增上线品牌服装站，拥有一群时刻关注时尚潮流的职业买手，技术精湛的服装生产从业者，只为追求更高品质，提供更好的服务。
