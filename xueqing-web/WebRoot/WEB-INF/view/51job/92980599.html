软件技术总监

长春-南关区 1-2万/月

吉林省联宇合达科技有限公司

职位信息

岗位职责：
1、负责组织建立公司软件质量管理体系，实施质量检查与监控。
2、不断优化公司的软件技术管理和项目管理流程，对公司过程改进提供支持；
3、负责重大技术项目的决策及方案审核，指导并参与核心代码的书写，组织解决项目开发过程中的重大技术问题；
4、负责指导、处理、协调和解决公司项目中出现的技术问题，保证项目的正常进行，确保公司项目计划按时完成；
5、及时掌握软件行业技术前沿动态，完成产品技术创新的可行性分析报告；
6、负责内部组织的建设和管理，建设技术团队，参与招聘、考核、培训等。
任职要求：
1、本科及以上学历，计算机相关专业，3年以上开发经验，5年以上的软件项目开发经验，3年以上团队管理经验，有丰富的软件队伍组织管理、协调经验；
2、熟练掌握软件工程的基本原理，能够规范地撰写软件分析和设计方案；
3、熟练掌握VS2005/2008,C#,JAVA、AS等开发工具和语言，有WEB应用系统开发经验，并多次担任项目核心模块的开发工作；
4、精通JAVA技术、SSH2、SSH、SpringMVC等架构设计，能够独立设计系统架构，设计产品线，攻克JAVA技术的难题；
5、精通Oracle、SqlServer数据库，能熟练操作linux、Windows操作系统，有较大型项目管理经验，技术全面，擅长管理；
6、有BI或大数据经验优先；
7、精通软件开发项目管理流程和软件研发技术，掌握本行业市场特点和开发过程，具有CMMI质量管理经验者优先；
8、精通C/C++/JAVA/PHP等主流体系架构之两种架构和设计模式，熟悉大数据和云计算相关技术理论，具有较强的架构和设计经验，对现有大数据、云计算主要产品的相关技术有深刻研究和了解；
9、精通数据库设计、分析、代码编写和调试，熟悉MySql/SqlServer/Oracle三种数据库架构中任一种的具体操作；
10、有智能交通领域大数据处理、实时监控以技术方面经验者优先，有MongoDB / Hadoop / Hbase / Cassandra / Storm / Spark / Hive 等系统的开发经验者优先。
待 遇： 具体薪资
公司福利： 双休+五险一金+餐补+话补+交通补助+年底奖金
           国家法定节假日休息+公费旅游+免费体检。
