职位信息

岗位职责：
1、对大数据生态带来的新型的商业运作模式进行研究，可以结合城市、区域和行业的发展战略制定产业促进的建议和商业计划书；
2、负责公司大数据、云计算和智慧城市类的投资、运营合作类项目的策划，对项目进行投资可行性分析，为投资决策提供依据；
3、配合销售经理完成与用户的大数据技术交流、技术方案宣讲等工作；
4、日常用户沟通、资料共享、技术协调等工作；
5、指导团队其他成员日常工作，确保团队工作目标的达成；

任职资格：
1、35~40岁，形象佳，普通话，大学本科8年以上工作经验或硕士研究生5年以上工作经验，3年以上大数据、云计算或者大数据分析、BI等方面的相关工作经验，2年以上团队管理工作经验；
2、有撰写大数据顶层设计方案的能力，从事过软件行业、智慧城市、BI类和云计算解决方案、技术或咨询岗位者优先；
3、具有五年以上大型国内外咨询、软件企业管理岗位从业经验者优先；
4、具有五年以上大型金融机构PE、VC项目投资实操经验，熟练掌握投资运作和项目操控技巧，具有成功案例及项目资源者优先；
5、思维清晰，学习能力强，能承受一定的工作压力；


能力要求：
1、具有行业内知名大数据、智慧城市售前解决方案管理岗位者优先考虑；
2、熟悉项目建议书、可行性研究报告、商业计划书、投标方案等各种文档规范及管理规范；
3、优秀的表达能力，能流利地演讲PPT等讲述材料；
4、具备较强的分析和解决问题的能力，有较强的沟通协调和组织管理能力，具备较高的职业操守和道德修养，具备较强的责任心和进取精神；
5、有高度的集体荣誉感与责任感，能体现出公司的良好形象。

职能类别： 咨询总监 售前/售后技术支持经理

关键字： 大数据 智慧城市 云计算 解决方案 顶层设计 咨询

公司信息

    广东粤数大数据有限公司是2015年由广东省广业资产经营有限公司联合国内知名大数据公司北京九次方及广东中泰集团成立，建立于国家“互联网+”行动计划以及“十三五”大数据战略的基础之上，致力于推动政府数据公开，挖掘各个产业的大数据价值及大数据应用。粤数大数据公司充分利用股东在大数据行业技术和战略优势，致力于推动政府数据公开，整合政府数据资源，建设大数据资产运营平台，挖掘各个产业的大数据价值及大数据应用，服务于政府及市场主体，推动数据成为政府及企业最重要的资产。
    广东粤数大数据公司拥有大数据挖掘技术、数据清洗、企业数据采集、数据建模、数据标准顶层设计、数据可视化技术方面的大数据全产业链人才。目前专利及软著等专业资质已达十余项。
