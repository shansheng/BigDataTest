职位信息

职位名称：解决方案工程师/全球服务部
地点：北京（全职）

工作职责：
1、按时交付代码或测试用例，提供必要的现场调试；
2、分析代码问题并解决。
3、及时向项目经理汇报；
4、与全球服务团队和其他团队紧密合作，例如销售团队。
5。根据需要提供测试服务。
6、可能分配的其他职责。

工作要求：
1、计算机科学或通信工程专业、网络工程的研究生。
2。软件设计经验和编码语言，如java、TCL、C／C++，PHP。
3、数据库配置和编程经验，如：SQL
4、熟悉Linux，了解Linux进程、内存管理、文件系统以及物理和虚拟网络知识；
5、协议和技术知识，特别是IP，TCP，RIP，OSPF，ISIS协议，HTTP，FTP；
6、必须有良好的团队合作精神。
7、良好的英语口语和书面表达能力
8、愿意出差

任职资格（学历/专业/证书）
1、硕士研究生，数据通信或相关专业；
2、良好的沟通技巧；
3、积极主动的团队合作精神；
4、较强的学习能力；
5、具有CCNP /SCPA相关认证


作为一家相当低调的外企，小伙伴们可能很少听过我们公司。但是通信行业内的人都知道，我们的实力超强。思博伦通信是全球领先的通信测试仪表及测试方案提供商, 拥有全面的测试解决方案，这些解决方案能够帮助客户顺利地开发和部署下一代网络技术及应用，是数据中心、云计算、SDN/NFV、虚拟环境、高速以太网、4G/5G网络及用户体验、卫星定位等测试领域的领导者。 思博伦通信的运营总部设在美国加利福尼亚州，在全球30多个国家及地区设有分公司或分支机构。思博伦通信公司(Spirent Communications plc) 的股票已在伦敦证交所上市（交易代码：SPT) 和纽约证交所(代码：SPM；CUSIP编号：84856M209) 挂牌交易。

【你将收获】：
1. 相关技能的强化培训和挑大梁的项目实练机会。技术大牛全程指导，满满的干货。
2. 个性超级好的同事领导，工作氛围良好。与世界各地同事的交流合作机会。
3. 外企工作流程统统知晓，对未来求职的公司定位会有明确目标。

关于思博伦通信
思博伦通信是全球领先的通信测试仪表及测试方案提供商，致力于帮助世界更快、更好地通信。
思博伦通信提供先进的性能分析与服务保证解决方案，从有线到无线再到卫星通信，思博伦拥有全面的测试解决方案。这些解决方案能够帮助客户顺利的开发和部署下一代网络技术，如宽带服务、IP电话、3G、Web应用以及安全测试，是以太网、VoIP、VPN、Triple Play、CDMA技术和应用测试的领导者。
思博伦通信是下一代通信网络的测试专家。我们的实验室测试解决方案已被世界诸多领先的通信设备提供商用来评估新技术和设备的性能。思博伦的产品应用在网络设备制造商研发实验室中，也被独立评估组织、网络运营商和网络提供商用于评估网络设备的性能。我们的技术专家为全球主要的通信标准组织提供有价值的咨询服务，帮助大型公司妥善保护和管理自己的网络。目前，我们正在帮助测试亚洲、欧洲和北美地区首批IPv6网络的部署。
思博伦通信的运营总部设在美国加利福尼亚州，在全球30多个国家及地区设有分公司或分支机构，并在美国、加拿大、英国、爱尔兰以及中国设立了研发机构。欢迎访问本公司网站。

关于思博伦通信中国
思博伦通信在中国的北京、上海、广州设立了代表处，并在北京成立了“思博伦通信技术服务及研发中心”，主要为电信运营商、通信设备制造商、大型企业、研究开发机构和评测实验室提供领先的通信测试设备、专业化测试技术服务以及相关测试软件与脚本研发，为网络设备的开发、技术研究、入网测试、设备选型、网络运维、网络优化及媒体公开测试提供不可或缺的工具。

Industry Experts
We pride ourselves on being experts on next-generation communication networks, devices and applications. Our engineers provide valuable counsel to many of the leading communication standards organizations. We pioneered testing of Ethernet networks, IP Telephony and VoIP, VPNs, TriplePlay, CDMA applications, and Location Based Services. Broadband Networking: DSL, Gigabit Ethernet, and IP
Convergence: VoIP, IP VPNs, IPTV
Next-Generation Internet: IPv6
Wireless: CDMA, UMTS, Location Based Services
Enterprise Networks: load testing, system performance, network security
Satellite Navigation: GPS, GLONASS, Galileo
