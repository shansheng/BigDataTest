技术运维工程师

广州-黄埔区 6-8千/月

广东赛百威信息科技有限公司

职位信息

岗位职责：
1、负责网站系统、数据库系统的集群配置、运维、服务监控分析、故障排查，以及紧急情况下的应急处理；
2、主持安装、配置、维护、测试生产环境，包括：网络配置、各类服务器搭建、数据库部署、WEB服务器端的部署；
3、负责系统（软件、数据库等），openstack，xen，虚拟化，分布式等架构的设计与分析；
4、分析硬件和软件集成后系统层面的问题，并提出解决方案；
5、研究分析已上线各类生产系统的应用服务体系结构及部署，发现潜在问题，制定系统调整和优化方案，实现性能优化，提高系统的健壮性和效率。
6、完成上级交付的其他事宜。

任职资格：
1、国家正规院校毕业，计算机相关专业，大专或以上学历；
2、2年以上系统管理工作经验；
3、熟悉LINUX和WINDOWS SERVER 2008操作系统日常维护、优化；
4、熟悉apache，tomcat等Web应用服务器部署、优化；
5、精通Shell/Perl/python的一种；
6、熟悉oracle的安装、配置、备份等；
7、熟悉负载均衡技术、集群技术和缓存技术；
8、了解互联网应用，对海量数据运维有浓厚兴趣，熟悉openstack、xen、rhev、ceph等云计算相关技术优先；
9、良好的沟通技巧和团队合作精神，正直进取、主动思考、有上进心。

职能类别： 技术支持/维护工程师 售前/售后技术支持工程师

关键字： 技术支持 系统维护 客户管理 系统支持 ITIL analyst it engineer

公司信息

广东赛百威信息科技有限公司（以下简称：赛百威）成立于1996年，是领先的互联网+服务提供商。作为最早从事软件开发的企业之一，赛百威迄今已有20+年经验积累，为宝洁、玛氏、屈臣氏、富力、万科等世界500强企业及政务部门成功提供了互联网化服务，并赢得广泛好评和信赖，成为客户长期的合作伙伴。
赛百威总部位于广州，在苏州、上海、武汉均设有分公司，服务踪迹遍布全国，并向亚太地区辐射，核心的技术团队由国内第一批从事互联网技术的专业人才组成，有多年的实战经验。
赛百威立足于行业洞察，以软件研发为核心，以客户需求为导向，在快消、地产、政务、厂内物流、连锁电商五大核心行业发力并取得重大突破。其中在智慧社区领域，赛百威成功打造了彩生活、合生活、国光2025PARK、富力等多个业界标杆；在互联网+企业应用上，赛百威发布了魔鸽移动设备管理平台、E-learning云学堂平台、workflow工作流平台等，持续为企业创造最大价值，积极推动社会信息化与数字化运作。
从2012年起，赛百威与兄弟公司广州博士科技展开深度合作和资源共享。广州博士俱乐部拥有上万名高级博士人才，赛百威借力博士科技的人才优势、研发优势和平台优势，将软件的研发与推广推向更高的层次。
为了更好地顺应市场发展及提供高质量的互联网+服务，赛百威重视自主创新不断加大研发投入，从软件定制逐步迈向产品化、平台化及开放运营的发展方向。现已自主研发出一系列创新型互联网+应用，并取得40多项专利及著作权。
融合创新，助力成长。赛百威将矢志利用创新的互联网技术推动社会的发展与变革，提升企业的核心竞争力，促进社会的高效运作。


丰富的企业文化生活

3、丰富多彩的员工组织：篮球协会、足球协会、羽毛球等体育组织；生日party、聚餐、联谊活动、户外拓展等亲情组织；
4、每年组织年度旅游。

3、公司设有餐厅、员工休息室、活动室，并为员工提供餐补、差旅补贴等；

7、公司致力于为员工提供发挥个人才能的舞台，良好的职业发展平台，对合适人才提供优厚薪资福利待遇；
8、上下班有交通班车接送。

随着公司各项业务的迅速扩展，期待有理想有追求的精英加盟。
