技术架构师--物联网项目

上海-徐汇区 3-4万/月

上海天坊信息科技有限公司

该公司所有职位

民营公司   |  50-150人   |  计算机软件,互联网/电子商务

申请职位 收藏 竞争力分析
8-9年经验 本科 招1人 02-08发布

职位信息

岗位职责
?支持售前做相关物联网项目的需求分析、架构定义、撰写解决方案、编制客户竞标方案、设计方案以及客户技术研讨会。
?从技术角度为客户提供不同的物联网 /云平台建议和系列技术服务
?指导内部或外部开发团队进行解决方案开发
?给客户提供物联网的技术培训与技术支持

岗位要求
?软件工程或计算机相关专业本科以上学历，10年以上工作经验
?8年以上的物联平台及应用开发经验或云计算平台开发经验，至少3年的架构师经验
?精通Java或C++等编程技术，熟悉Linux系统，独立开发过项目或产品
?熟悉软件开发流程和相应规范
?了解传感器、移动通信网架构等基本知识与原理，熟悉RFID、WiFi、Zigbee、LoRa、NBloT等传感器关键通信组网技术，并具有相关技术的综合组网经验
?熟悉主流的云平台（阿里云、AWS、Azure等）开发和扩容，SaaS/PaaS/IaaS
?对微服务应用程序架构有丰富经验
?丰富的企业移动接口架构的经验(iOS / Android,Xamarin等)
?熟悉物联网相关通信协议。具有大数据分析经验者优先
?至少一个成功的大型物联网项目开发和架构经验
?良好的英语读写能力

加分条件
?熟悉至少两种云平台者优先，例如阿里云平台, AWS IOT, IBM Watson, IBM Bluemix, MS Azure, GE Predix, PTC ThingWorx, Google Cloud, Cloud Foundry等
?有嵌入式软件开发或嵌入式操作系统开发经验者优先
?对加密和安全方面的架构有经验者优先
?熟悉国内外主流传感设备厂商并有合作者优先
?有IBM、Google、Amazon、BAT等国内外大型厂商的项目架构经验者优先
?熟悉Docker和容器技术者优先
?有敏捷开发的项目经验者优先
?英语读写熟练，有海外工作经验者优先
?能接受一定的出差安排；

项目背景
智慧城市项目，例如整个城市的物联网电梯、物联网管道、物联网安防等类型项目，做到实时监测、预警、追溯排查和前瞻性运维等功能。

职能类别： 系统架构设计师

公司信息

    上海天坊信息科技有限公司坐落于上海市徐汇区淮海中路1010号, 5A级豪华办公写字楼，紧邻襄阳公园和地铁1号、7号、10号线、12号线。公司拥有雄厚的技术实力，是服务于政府、大型企业的专业化公司。业务范围涵盖上海、广东、湖北、北京等全国各地和澳洲，涉及文化、体育、餐饮、供应链等诸多行业和领域。
    公司拥有来自互联网知名公司及名牌院校出身的职业精英，不断推出平台化、互联网、移动互联网、VR等产品。先进的技术和快速的新产品研发为公司员工带来了前所未有的挑战和机会。公司氛围年轻，快速发展，成长空间巨大。
    公司前期项目包括食品安全追溯系统、体育互联网O2O平台、行业智能管理系统等等，行业领先，影响力大。项目稳定持续，发展前景好。

    我们为每一位辛勤耕耘的员工提供有竞争力的丰厚薪资，每年都有升职加薪机会。
1、如果你参加项目开发实施，你会得到丰厚的项目奖金；其他包括五险一金、节日福利、带薪年假、团建活动等等。而且你还将拥有全面的、广阔的职业发展空间，包括横向的、纵向的晋升通道，系统完善的培训体系。
2、宽敞、舒适的办公环境，公司地处市中心地段，给你家一般的温暖。
3、温馨的人际环境，可爱的工作伙伴，公司俊男美女云集，干活特别有动力。
4、我们不定期的组织各项员工活动，丰富大家的业余生活。
    优越的工作环境，富有诱惑力的薪酬体系，诚邀超有激情的你加入我们！希望你能够在我们的职业平台上充分发挥专业优势，获得最具成就感的职业生涯，同时，相信通过你和团队的共同努力，我们将共同开启一段辉煌的职业旅程。
    只要您足够优秀！这里将是实现您事业成就与人生乐趣的舞台！
