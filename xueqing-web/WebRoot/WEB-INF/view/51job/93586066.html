职位信息

工作职责：
负责IoT App开发，Cloud Solution的构建，以及IoT领域内的智能化服务。

岗位要求：
计算机相关专业，满足以下至少一种条件：
1. 熟练使用JAVA、C++、JS中的一种语言；了解常见的通讯协议如HTTP, MQTT, CoAP等.
2. 有Web服务端项目开发经验, 对服务端安全，故障诊断，和高可用运营有一定经验。
3. 熟悉microservice架构理念，了解常见的分布式RPC框架如dubbo， thrift， pb.
4. 了解智能化服务，如基于用户行为的Rule推荐和推理，IoT边缘计算包括过滤，预测，预处理等。

附加分项：
1. 熟悉一种IoT云计算平台，如Azure IoT, AWS IoT或者其他IoT云端解决方案。
2. 熟悉开源社区组织运作，有OCF，mBed、Thread、HomeKit相关经验，
3. 有国内IoT平台如JD，Aliababa，Tencent，Haier接入经验。或者熟悉liteos，JoyLink等软件方案。
4. 有本领域相关的专利，论文和工业上物联网从业经验。

三星电子（中国）研发中心2004年成立于南京，是三星电子全资海外研究所。
公司在三星电子的国际化研发战略指引下，承担着面向国际和中国市场的数字化领域系统解决方案和前沿技术研究开发的责任，秉承三星“人才第一、最高志向、引领变革、正道经营、共存共赢”的价值观，在发展的进程中，履行着我们的使命：任用优秀的中国人才，结合中国技术标准和市场特点，通过持续的研究开发，为客户提供最好的研究成果和技术解决方案。

公司目前专注于核心应用平台和关键应用技术的先行研究和开发，致力于Linux平台和Android平台手机软件开发、数字电视应用系统软件等专向研究开发以及Web Service、内存数据库、交互软件等开发。

随着公司业务的迅速扩大，现正在招募大批新员工。 公司将为新员工提供良好的培训，完善的薪资与福利。
有竞争力的薪资体系；公司严格按照国家法律规定缴纳社保和公积金，支付加班费；提供本人及子女在内的补充商业医疗保险，实行人性化的弹性工作制，差异化的带薪年假制度。

我们衷心希望优秀的人才加盟三星电子（中国）研发中心，为壮大本地研究开发实力，为实现在中国建设第二个三星的愿景，为服务于更美好的未来世界而共同努力。
