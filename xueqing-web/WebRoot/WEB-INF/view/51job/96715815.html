极数宝之名：

“极数知来之谓占，通变之谓事。”取自【易经·系辞】
利用极数之规律从而推演万事万物就可以称之为“占”，通晓阴阳变化采取相应措施用之于民造福于民就是事务，就是人道。
极数配一“宝”字，将赋予新的划时代的含义：Find Value In Data。

极数宝之事：
在大数据应用雷声大雨点小的当下，极数宝公司默默地闯出了一条技术与市场完美结合的新路。利用其核心技术在税务系统中大显身手，充分挖掘互联网数据价值，形成税务信息情报“天网”。有力地推动了税务部门转变征管方式，推进了税收治理体系和治理能力现代化。对各级政府部门、企事业单位在互联网情报收集应用上具有很好的借鉴作用，应用前景广阔。

极数宝之求：
经过自有资金和在中国税务领域大数据服务营收的双重投入，极数宝的发展已经渐渐进入正轨，并驶入高速发展的快车道。但随之而来的是人才的严重不足尤其是高端研发领军才人和高端业务领军人才。
